EMF Forms is a subcomponent of EMF Client Platform (since 1.2.x) and is hosted on the same update site. To install the EMF Forms, you need a running version of Eclipse. The simplest way to get started is to use the latest modeling edition. To get started, please use the EMF Forms SDK Feature.

1.3.x Milestones

Please copy this URL:
http://download.eclipse.org/ecp/releases/releases_13/130_M4
  1. Navigate to Help → Install New Software
  2. Paste the copied URL into the text field label with "Work with" and press enter
  3. Select the features, you want to install (if you are unsure: the "EMF Forms SDK"). Please also visit our documentation for more information
  4. Accept the license agreement and finish the installation
