To install the EMF Client Platform, you need a running version of Eclipse. The simplest way to get started is to use the latest modeling edition.
Otherwise install the EMF Forms SDK Feature from the Update Site (NOT the Target Platform Update Site).

1.17.x Release Candidates

When updating to this version, please have a look at our migration guide: EMFForms Migration Guide.
Please copy this URL:
http://download.eclipse.org/ecp/releases/releases_117/
  1. Navigate to Help → Install New Software
  2. Paste the copied URL into the text field label with "Work with" and press enter
  3. Select the features, you want to install (if you are unsure: the "EMF Forms SDK"). Please also visit our documentation for more information
  4. Accept the license agreement and finish the installation
Please copy this URL:
http://download.eclipse.org/ecp/releases/releases_target_116/
